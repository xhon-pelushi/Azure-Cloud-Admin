# Azure Cloud Administration

Azure cloud infrastructure management project covering Entra ID (Azure AD) configuration, MFA, RBAC, resource provisioning automation, and ARM templates.

## Overview

This project demonstrates Azure cloud administration skills:
- Entra ID (Azure AD) configuration and management
- Multi-Factor Authentication (MFA) setup
- Role-Based Access Control (RBAC)
- Automated resource provisioning with Azure CLI and PowerShell
- ARM templates for infrastructure as code
- Cloud security and cost management

## Components

### Identity Management
- Entra ID user and group management
- Conditional Access policies
- MFA enrollment and enforcement
- Application registrations

### Infrastructure Automation
- ARM templates for VM, Storage, and Network resources
- Azure CLI scripts for resource provisioning
- PowerShell automation modules

### Security
- RBAC role assignments
- Security baselines
- Compliance monitoring

### Cost Management
- Cost analysis and reporting
- Budget alerts
- Resource tagging strategies

## Requirements

- Azure subscription
- Azure CLI 2.0+
- PowerShell Azure modules
- Visual Studio Code (for ARM templates)
